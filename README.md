# Sketchpacks

### What is Sketchpacks?

> Sketch Plugin Management

Sketchpacks is a macOS menubar app that empowers Sketch users to manage, auto-update, and share Sketch plugins.

### Compatibility Note

Sketchpacks supports Sketch 40 and above.
